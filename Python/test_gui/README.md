## WHY
    众所周知，个人编写任何项目时，由于无法进行有效而全面的测试，所以性能无法保证。
    那么是否拥有一个可视化用户界面（GUI）就成为一大卖点。

    而任何语言自带的GUI库无疑是一块旧的“硬骨头”，而且文档阅读不易。
    若单纯通过阅读博客，费时费力不说，还可能存在“知识传递错误”。

    第三方GUI库有活跃的社区，并且仍在实时同步更新。最重要的是**文档详细易阅读**。

    本仓库旨在学习第三方Python GUI库，代码主要是一些文档内容的实践。
